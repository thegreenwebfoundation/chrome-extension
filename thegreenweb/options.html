Highlight Green Search Results:

The Green Web Search adds the different green web icons to search results for the Google, Yahoo, Bing and Ecosia search engines. Check out the add-ons page for information on the different icons and how it works.

The Green Web Ethical filter:

If you enable the green web search, you can also enable filtering, meaning that any grey results will not be shown anymore.

The Green Web for all external links:

This web extension can also be used to check all external links on a page. All links to other domains that are greenly hosted will receive a green underline. Hovering over those links will show a small popup with more information.

To check that page or link if is hosted green, this extension needs to send its domain name against to The Green Web Foundation Greencheck API. No other information is sent to other than this domain name, and these features can be switched on or off at any time. If you want to know more, please see our privacy statement for more details.
